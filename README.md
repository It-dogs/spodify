# Spodify

#### Description
This is a spotify clone project called spodify developed by YinHk(Ken), using the sportify API. 

#### Tech stack
Project is being build with the following tech:
- ReactJS & redux, 
- MUI, 
- JSS, 
- styled component,
- framer motion, 
- react-router, 
- swiper.js 
- mongodb
- firebase  
etc

Project is still ongoing, started from June 2022.
